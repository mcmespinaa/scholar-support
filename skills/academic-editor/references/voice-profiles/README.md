# Voice Profiles

Place author voice profile files here as `.md` files. The academic-editor skill (Pass 3: Voice Editor) uses these to maintain consistent author voice during editing.

## Creating a Profile

Each profile should describe:
- **Writing style**: sentence length, structure preferences, transitions
- **Academic register**: level of hedging, formality, directness
- **Spelling conventions**: British English, American English
- **Formatting habits**: em dash usage, parenthetical style
- **Supervisor influence**: if applicable, how supervisor feedback shapes the voice

## File naming

Use `author-name.md` (e.g., `maria-espina.md`, `john-smith.md`).
